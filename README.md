# kc-cask
Knowledge Commons Create A Site Kit contains files and instructions for site templates.
The files in this repository can be used to populate a WordPress site with template pages and sample content on Knowledge Commons or other WordPress instances.

## Basic Steps

1. Establish a Knowledge Commons account (if not already done)
2. Create a new site on Knowledge Commons once you are logged in
3. Download the template of choice from this repository
4. Follow the importing instructions below. 

## Importing instructions

1. Template should be downloaded to your computer and you'll need to know where it is to upload
2. Go to your site and make sure you are logged in. 
3. In the top black ribbon you'll see a little house icon and your site name next to it, mouse over that (don't click) and choose Themes
4. On the Themes page make sure that the theme "Twenty Twenty-Four" is active, if it's not find it on that page and activate it.
5. Next, on the left side there is a menu and toward the bottom you'll see Tools with a wrench icon, Click on this and then it will open more menu options below it, Import should be at the top right under Available Tools.
6. Click Import and the import screen will load, from there click Run Importer and the Import upload page will load.
7. Click the Choose File button and browse in your computers files for the template file you downloaded and click Open
8. Now click Upload file and import in WordPress. The Assign Authors page will come up showing content authors of the template. In the "Assign posts to an existing user" drop down menu for each person choose yourself. Then click the "Download and import file attachments" checkbox  and press Submit.
9. It will take a moment as it imports the content and then will give you a confirmation message "All done. Have fun!" if it was successful

You should now be able to edit your site, modifying pages and creating new ones.
